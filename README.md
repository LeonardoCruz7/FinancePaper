
# FinancePaper 

---
## Proposta

O FinancePaper é um aplicativo desenvolvido em Swift onde sua principal tarefa é armazenar estoque e valor dos produto, adicionar valores de serviço e mão de obra e utilizar o Markup como métrica para dar um valor que consegue acobertar todas as despesas do usuário.

---

## Funcionalidades

- [x] **Persistencia de Dados:** Usa o SwiftData para registrar as despesas e estoque do usuário.
- [x] **API Rest para consulta de valores de produtos:** O usuário pode pesquisar valores de materia-prima de seu negócio verificando em qual loja compensa mais a sua compra.
- [x] **Widget:** Widget de lançamento rápido de despesa.
  

---

## Tecnologias 

- **Linguagem Principal:** Swift
- **Frameworks & Libs:** SwiftUI, SwiftData, CloudKit
- **Banco de Dados: SQLLite
